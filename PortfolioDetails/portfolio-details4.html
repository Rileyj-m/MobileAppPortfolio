Percolation Description

For percolation, we had to write a program to estimate the value of the percolation threshold via Monte Carlo simulation. To achieve this, we had to implement a functional percolation data type, making use of WeightedQuickUnionUF and QuickFindUF. To do this, I created a class called Percolation, which implements the functional data type. Then I created a grid of size NxN, and created a WeightedQuickUnionUF object to represent the grid. Based on the simulation, I would open a sight and add it to the grid. In the end I would check if all the sites were connected, and display the result. This was my first introduction to implementing functional data types in Java. I learned how to implement a functional data type in Java, and really grasped the java language.
