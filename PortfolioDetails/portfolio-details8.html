Grocery Shopping List App V1 Description

For this first project version, we had to create an Android app that receives user input and implements a RecyclerView in order to perform some meaningful task using that input. We had to implement Material styling in our app. And create appropriate packages.


Grocery Shopping List App V2 Description

For this second project version, my project had to incorporate user input, LiveData, and a ViewModel, similar to the LunchTray app. Using a RecyclerView was optional, but I ended up using one. My app also had to navigate between two or more fragments by implementing a NavGraph and NavHostController.
