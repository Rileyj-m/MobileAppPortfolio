Sierpinskys Triangle Description

This is a python program that displays a Sierpinski's Triangle. The idea behind this triangle is a fractal based on a triangle with four equal triangles inscribed in it. The central triangle is removed and each of the other three treated as the original was, and so on, creating an infinite regression in a finite space. The way that I solved this coding problem was to use a recursive function to draw the triangle. This was expected from the teacher because I hadn't learned recursion before this project. So after this project was done not only did I learn how to read and understand API's but I also learned how to use them in my own programs. I also learned the fundementals of recursion and how to use it in my own programs.
